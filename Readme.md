# Password Generator 🔐

A command-line password generator built in Python. Choose your password length and character types to generate a secure password.

## Features
- Custom password length
- Choose character types:
  - Digits (0-9)
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Symbols (!@#$...)
