# Password Generator 🔐

A command-line password generator built in Python. Customize your password length and character types every time you generate a new password.

## Features
- Custom password length per generation
- Choose character types each time:
  - Digits (0-9)
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Symbols (!@#$...)
- Input validation for length (try/except)
- Must select at least one character type
- Generate another password with new length and characters each time

## How to Run

Make sure Python is installed, then run:
python password_generator.py
